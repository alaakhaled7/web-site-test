Definition of programming

Programming is the implementation of logic to facilitate specific computing operations and functions, and occurs in one or more languages, which vary by application, field, programming model, and writing method. Programming language semantics and syntax are used when creating an application, hence programming requires knowledge of application areas, algorithms, and programming language expertise. The logic of a programming language varies by developer, and good code can be evaluated by factors such as:
Durability: Focuses on the sustainability of the program, regardless of bugs or incorrect data.
Reliability: Focuses on correct design and algorithm implementation.
Efficiency: Focuses on memory, hardware, or other characteristics used to improve software.
Readability: Provides appropriate documentation and indentation, which provides insight to other software developers or designers.

programming languages :

A programming language has a vocabulary and a set of grammar rules, to guide the computer, to perform specific tasks, each programming language contains a unique set of keywords, and a special syntax to organize the program instructions.

Types of programming languages:

There are dozens of programming languages ​​used, in terms of high-level and low-level languages, and the following are the most important programming languages:
Java: A general purpose, object-oriented, high-level programming language with many features that make it ideal for web-based development.
Ruby: An open source, object-oriented scripting language that can be used independently or as part of the Ruby on Rails framework.
HTML: The standard markup language used to create web pages, it ensures the correct formatting of text and images (using tags), used by web developers, technical editors, email designers and software engineers.
JavaScript: a client-side programming language that runs inside the client's browser, and processes commands on a computer instead of on a server.
C: a structure-oriented programming language, mostly used for developing low-level applications.
C ++: a general purpose programming language, object-oriented, mid-level, extension of C language.
Objective-C: A simple, general-purpose language, using a borrowed passer-by message system from SmallTalk, used by iOS and Android app developers.
PHP: An open source programming language designed, to create dynamic web pages that work effectively with databases.
SQL Language: a database query language, (not a development language) that allows adding, accessing, and managing content in a database.
Swift Language: Apple's latest open source multi-paradigm programming language for iOS and OS X applications, and includes an advanced translator, debugger, and framework infrastructure.
Python: a sophisticated programming language that is interpreted and directed towards an object, and built on flexible and strong semantics.
